3 拖动示教

拖动教学

drag_teach_save()

开始录制并拖动教学点。

  • 注意：为了显示最佳运动效果，录制时间不应超过90秒

drag_teach_pause()

暂停采样。暂停采样并再次启用机械臂

drag_teach_execute()

开始拖动示教点，仅执行一次。

＃ 案例


from pymycobot import Mercury
import time
ml = Mercury("/dev/left_arm")
mr = Mercury("/dev/right_arm")

ml.power_on()
mr.power_on()
# 左臂开始轨迹记录
ml.drag_teach_save()
# 记录时长为10秒
time.sleep(10)
# 停止记录
ml.drag_teach_pause()
time.sleep(1)
# 开始执行记录的运动轨迹，仅执行一次
ml.drag_teach_execute()
